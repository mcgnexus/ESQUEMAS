# StudyBuddy AI - Generador de Esquemas Inteligente

Transforma tus apuntes en esquemas visuales y mapas mentales utilizando Inteligencia Artificial.

## Características

- **Generación Inteligente**: Sube fotos de tus apuntes o PDFs y obtén esquemas estructurados automáticamente.
- **Múltiples Formatos**:
  - **Mapas Mentales**: Estilo jerárquico (Buzan).
  - **Mapas Conceptuales**: Conexiones cruzadas y explicaciones (Ausubel/Novak).
- **Editor Visual Interactivo**:
  - Edita nodos, reorganiza la estructura y personaliza colores.
  - Atajos de teclado y navegación accesible.
  - Diseño responsivo para móviles y escritorio.
- **Exportación Versátil**:
  - PDF (listo para imprimir).
  - PNG (imagen de alta resolución).
  - Markdown (para notas).
  - JSON (copia de seguridad).
- **Gamificación y Estadísticas**:
  - Rastrea tu tiempo de estudio.
  - Visualiza tu progreso y sesiones.
- **Accesibilidad**:
  - Alto contraste y soporte para lectores de pantalla.
  - Navegación completa por teclado.
  - Modo oscuro y claro.

## Tecnologías

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS, Lucide React
- **Visualización**: ReactFlow, Dagre
- **Utilidades**: jsPDF, html2canvas, modern-screenshot

## Instalación y Uso

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/mcgnexus/ESQUEMAS.git
    cd ESQUEMAS/esquemas-app
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    ```

3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```

4.  Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
